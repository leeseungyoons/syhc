# Sports Management Streamlit App

Streamlit을 활용해 구현한 **스포츠 종목 관리 미니 프로젝트**입니다.  
야구, 축구, 농구 등 다양한 스포츠 종목 데이터를 카드 형태로 조회하고, 사용자가 직접 **추가 / 수정 / 삭제 / 검색 / 필터링**할 수 있도록 구성했습니다.

이 프로젝트는 대학 실습 및 개인 학습 과정에서 진행한 미니 프로젝트로,  
**Streamlit 기반 웹 UI 구성**, **세션 상태 관리**, **간단한 CRUD 기능 구현**을 직접 경험하고 익히는 데 목적을 두었습니다.

---

## 프로젝트 소개

이 앱은 스포츠 종목 정보를 직관적으로 관리할 수 있는 Streamlit 기반 웹 애플리케이션입니다.

사용자는 다음과 같은 기능을 이용할 수 있습니다.

- 스포츠 종목 목록 조회
- 종목명 검색
- 카테고리별 필터링
- 국가 검색
- 즐겨찾기 관리
- 종목 추가 / 수정 / 삭제
- JSON 데이터 다운로드

특히 야구(KBO, MLB) 데이터를 포함해 다양한 스포츠 리그 정보를 카드형 UI로 보여주며,  
간단한 관리 시스템 형태로 활용할 수 있도록 구현했습니다.

---

## 프로젝트 목적

이 프로젝트는 단순히 데이터를 출력하는 수준을 넘어서,  
사용자가 직접 데이터를 관리할 수 있는 **미니 서비스 형태의 웹앱**을 만들어보는 데 목적이 있습니다.

이를 통해 다음과 같은 부분을 학습했습니다.

- Streamlit 기반 웹 애플리케이션 구조 이해
- `st.session_state`를 활용한 상태 관리
- 폼 입력을 통한 데이터 추가 및 수정
- 카드 UI와 탭 UI를 활용한 화면 구성
- 검색 / 필터 / 정렬 기능 구현
- JSON 다운로드 기능 구현

---

## 주요 기능

### 1. 종목 목록 조회
등록된 스포츠 종목을 카드 형태로 확인할 수 있습니다.

### 2. 검색 및 필터링
- 종목명 검색
- 카테고리별 필터
- 국가 검색
- 즐겨찾기만 보기

### 3. 종목 추가
새로운 스포츠 종목을 직접 등록할 수 있습니다.

### 4. 종목 수정
기존 종목 정보를 선택해 수정할 수 있습니다.

### 5. 종목 삭제
등록된 종목을 삭제할 수 있습니다.

### 6. 즐겨찾기 기능
중요한 종목을 즐겨찾기로 관리할 수 있습니다.

### 7. JSON 다운로드
현재 종목 데이터를 JSON 파일로 다운로드할 수 있습니다.

## 실행 방법

### 1. 저장소 클론

```bash
git clone https://github.com/leeseungyoons/sports-management-streamlit.git
cd sports-management-streamlit
```

### 2. 가상환경 생성 및 활성화

#### Windows
```bash
python -m venv .venv
.venv\Scripts\activate
```

#### macOS / Linux
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. 라이브러리 설치

```bash
pip install -r requirements.txt
```

### 4. 앱 실행

```bash
streamlit run app.py
```

## 기대 효과

이 프로젝트를 통해 단순한 데이터 출력이 아니라,  
사용자가 직접 데이터를 관리할 수 있는 **실사용 형태의 미니 CRUD 웹앱**을 구현할 수 있었습니다.

또한 Streamlit을 활용해 다음과 같은 경험을 쌓았습니다.

- 상태 기반 인터페이스 구현
- 사용자 입력과 데이터 반영 흐름 구성
- 가벼운 관리형 서비스 개발 경험

- 현재는 로컬 세션 기반 상태 관리 방식이라, 앱을 새로고침하면 데이터가 초기화될 수 있습니다.
